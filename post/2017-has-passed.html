2017 已逝

Tagged as note
Written on 2017-12-31 15:26:20

2017 的目标实现了吗

先回顾一下 2017 年的目标:

  • Go 语言继续趁手

  • Kotlin 修炼, 捎带 Java 8

  • Python 3.6 特性加持

  • OCaml 把玩

除了 Go 语言继续趁手, 其他的真的没所谓了.

现实会给你以教训

2017 年不管对于工作还是生活真的让我涨见识了;

  • 好好活着;

  • 走出舒适区;

  • 野心需要技能匹配;

2018 年做好自己

  • 简单与优雅正交, 写清晰的代码, 做简单的事, Go 继续趁手;

  • 认准的事情就干, 不骄不躁;

  • 逼一下自己;
